# Anagrammas pārbaude

Šī Go programma nolasa no ievades divus vārdus. <br>
Tā pārbauda, vai otrais vārds ir pirmā vārda anagramma, t. i., vai abiem vārdiem sakrīt burtu (simbolu) sastāvs un katra burta skaits.<br>

Ja otrais vārds ir pirmā vārda anagramma, programma izvada "YES". Pretējā gadījumā tiek izvadīts "NO".
